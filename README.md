# The Code Lexicon

Utilize OpenAI to translate code from one language to another.

![Code Lexicon](./public/screenshot.png)

## To run Locally

**1. Clone Repository**

```bash
git clone https://github.com/mckaywrigley/ai-code-translator.git
```

**2. Install Dependencies**

```bash
npm i
```

**3. Run App**

```bash
npm run dev
```
